OpenGL

  • OpenGL exists since 17 years and it has under gone some radical changes.
  • We split OpenGL into old and modern OpenGL
  • Old OpenGL is considered deprecated
  • OpenGL ES 2.0/3.0/3.1 - Mobile
  • WebGL - Browser
  • OpenGL 3.3 - Similar to DX9
  • OpenGL 4.x - Similar to DX11
  • Modern OpenGL is still a state machine.
  • This means that we have global mutable state.
  • OpenGL is type unsafe. It's up to the programmer to don't make mistakes.
  • The OpenGL API is written in C.
  • Driver quality is dependent on the manufacturer (Nvidia, Amd, Intel etc)
  • We now have more control over the pipeline.
  • The programmable parts of this pipeline can be programmed in GLSL. (Shader)
  • We also have control over the data that we are sending to the GPU.
Some programmable shader stages:
  • Vertex shader
  • Fragment shader
  • Tessellation shader
  • Geometry shader
  • Compute shader
OpenGL Debugger:
  • Nsight - Nivdia and Windows only
  • APITRACE
  • VOGL
